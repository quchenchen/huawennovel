# 竖屏短剧知识库查询专家 - 详细示例

## 示例 1: 查询创作规范

### 输入

```
请查询以下知识：

竖屏短剧的剧本格式规范是什么？
```

### 输出

```
【竖屏短剧剧本格式规范】

一、基本格式
```
场次号 场景 时间 内外景

人物：（人物状态和动作）
人物：台词内容

【动作/场景描述】

人物：（人物反应）
人物：台词内容
```

二、格式要求
1. 单集时长：1-3分钟
2. 单集字数：300-800字
3. 场次数量：通常2-5场

三、剧本要素
1. 场景信息：地点、时间、内外景
2. 人物信息：角色、状态、动作
3. 台词内容：对话、独白、画外音
4. 动作描述：表演提示、场面调度
```

## 示例 2: 查询评估标准

### 输入

```
请查询以下知识：

竖屏短剧的评估标准是什么？
```

### 输出

```
【竖屏短剧评估标准】

一、核心爽点评估
- 评估维度：普世化程度、清晰具体化、节奏递进
- 评分标准：
  - 9-10分：极致爽感，强烈共鸣
  - 7-8分：良好爽感，有共鸣
  - 6分以下：爽感不足

二、故事类型评估
- 评估维度：强设定、经典+创新、极致化
- 评分标准：
  - 9-10分：类型创新，极具吸引力
  - 7-8分：类型经典，有创新点
  - 6分以下：类型老套，无新意

三、人物设定评估
- 评估维度：反差感、欲望明确、功能化
- 评分标准：
  - 9-10分：人设极致，记忆深刻
  - 7-8分：人设鲜明，有特点
  - 6分以下：人设平淡，无特色

四、人物关系评估
- 评估维度：强冲突、强绑定、权力结构
- 评分标准：
  - 9-10分：关系极致，张力十足
  - 7-8分：关系清晰，有张力
  - 6分以下：关系松散，无张力

五、情节桥段评估
- 评估维度：切入矛盾、情绪极致、视觉转化、悬念设置
- 评分标准：
  - 9-10分：情节极致，爽感十足
  - 7-8分：情节完整，有爽感
  - 6分以下：平淡无奇，无爽感
```

```
如何设计竖屏短剧的悬念钩子？

```

```
- 方法1：直接切入矛盾
- 方法2：设置视觉冲击

四、钩子技巧
```

### 输入

```
请查询以下知识：

竖屏短剧如何设置付费卡点？
```

```
【付费卡点设置技巧】

一、卡点位置选择
- 位置1：高潮前（3-5集）
- 位置2：转折前（7-8集）
- 位置3：大结局前（最后一集前）

1. 在情绪最高点设置
2. 在真相揭晓前设置
3. 在冲突爆发前设置

1. 悬念型卡点："真相即将揭晓..."
2. 冲突型卡点："大战一触即发..."
3. 转折型卡点："主角的命运将如何..."
4. 情感型卡点："他们能否走到一起..."

四、卡点效果评估
- 评估标准：付费转化率、观众满意度
- 优化建议：根据数据调整卡点位置和内容
```
