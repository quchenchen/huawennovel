# 已播剧集分析与拉片专家 - 详细示例

## 示例 1: 剧集整体分析

### 输入

```
请对以下剧集进行整体分析：

剧集名称：《都市爱情故事》
集数：20集
类型：都市情感剧

主要情节：林浅和陆远在广告公司从竞争对手发展为恋人的故事。
```

### 输出

```
【剧集分析报告】

一、剧集基本信息
- 剧集名称：《都市爱情故事》
- 集数：20集
- 类型：都市情感剧
- 风格：轻松温馨

二、故事结构分析
- 叙事结构：线性叙事，时间顺序发展
- 三幕式结构：完整符合
- 节奏把握：张弛有度，节奏适中

三、人物塑造分析
- 主角塑造：立体丰满，成长弧线完整
- 配角塑造：相对单薄，有待提升
- 人物关系：清晰但复杂度有限

四、情节设计分析
- 情节创新：传统套路，创新有限
- 冲突设计：冲突强度适中
- 反转设计：反转较少，惊喜不足

五、视听语言分析
- 场景设计：职场场景真实
- 镜头运用：中规中矩
- 音乐运用：贴合情节

六、市场表现预测
- 目标受众：25-35岁女性观众
- 竞争力：中等偏上
- 推荐指数：★★★☆☆
```

## 示例 2: 单集拉片分析

### 输入

```
请对第5集进行拉片分析：

第5集内容：
林浅和陆远在深夜加班时，聊起了彼此的过去。陆远透露他曾经是一个封闭的人，是林浅的出现改变了他。林浅被感动，两人之间的距离进一步拉近。
```

### 输出

```
【单集拉片分析报告 - 第5集】

一、本集基本信息
- 集数：第5集
- 时长：45分钟
- 核心事件：深夜加班谈心

二、场景分解
- 场景1：办公室深夜（开场）
- 场景2：办公室谈心（核心）
- 场景3：分别回家（结尾）

三、镜头分析
- 开场镜头：空镜头切入，营造深夜氛围
- 核心镜头：双人近景，展现情感交流
- 结尾镜头：分镜头处理，暗示各自心动

四、人物表演
- 林浅：外冷内热，细微表情到位
- 陆远：真诚坦率，眼神戏精彩

五、对白分析
- 台词风格：自然真实，符合人物性格
- 金句提取："是你让我学会了打开心扉"
- 情感表达：含蓄而深情

六、情节功能
- 推动关系：从同事向暧昧迈进
- 人物成长：陆远展示真实自我
- 情感高潮：深夜谈心的浪漫氛围

七、本集评价
- 情感浓度：★★★★☆
- 表演质量：★★★★☆
- 导演水准：★★★☆☆
```

## 示例 3: 五元素综合分析

### 输入

```
请对以下剧集进行五元素综合分析：

剧集：《古装宫斗》
主角：苏婉
情节：从秀女到皇后的历程
```

### 输出

```
【五元素综合分析报告】

一、题材类型分析
- 主导题材：古装宫斗
- 辅助题材：权谋、成长
- 创新程度：中等

二、故事梗概
苏婉从礼部尚书嫡女被选入宫，凭借聪慧得宠。皇后嫉妒陷害，苏婉被打入冷宫。在冷宫中，苏婉学会了宫廷斗争技巧。最终苏婉成功反击，成为皇后。

三、人物小传
【苏婉】
- 身份：尚书嫡女→秀女→妃嫔→皇后
- 性格：聪慧过人，善于学习
- 成长：从单纯到善于权谋

四、人物关系
- 苏婉↔皇上：君臣夫妻
- 苏婉↔皇后：敌对关系
- 苏婉↔太妃：师徒关系

五、大情节点
1. 入宫得宠
2. 遭陷被打入冷宫
3. 冷宫成长
4. 救驾重获宠爱
5. 揭露阴谋成为皇后
```
