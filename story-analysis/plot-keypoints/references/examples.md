# 大情节点分析专家 - 详细示例

## 示例 1: 三幕式结构 - 都市情感剧

### 输入

```
林浅是一名28岁的广告公司策划经理，从小城市来到北京打拼。在竞标一个重要项目时，她遇到了竞争对手陆远。

两人在竞争中针锋相对，但在合作过程中逐渐发现彼此的才华和魅力，从竞争对手发展成为恋人。

然而，陆远的前女友突然回国，试图挽回他。面对前女友的纠缠和家庭背景差异的考验，两人最终克服所有阻碍，走到了一起。
```

### 输出

```
【阶段一：初识与竞争（建立）】
- 林浅和陆远在项目竞标中首次相遇，针锋相对
- 两人成为竞争对手，在多个项目中展开较量
- 竞争过程中，双方展现各自的专业能力和个性特点

【阶段二：合作与相知（发展）】
- 重要项目要求两人合作，从对抗到磨合
- 在工作合作中发现彼此的才华和魅力，关系开始转变
- 两人逐渐了解对方，从竞争对手到合作伙伴再到朋友
- 感情逐渐萌芽，但双方都尚未明确表达

【阶段三：考验与圆满（高潮与结局）】
- 项目成功后，两人开始约会，感情正式确立
- 前女友回国，制造情感障碍和冲突
- 职场竞争和家庭背景差异带来新的考验
- 陆远明确拒绝前女友，向林浅表达真心
- 两人克服所有阻碍，最终走到一起
```

## 示例 2: 五幕式结构 - 古装权谋剧

### 输入

```
苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，她发现后宫中充满了勾心斗角，皇后和各妃嫔之间明争暗斗。

苏婉凭借自己的智慧和谨慎，在后宫中逐渐站稳脚跟。然而，皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。

在冷宫中，苏婉结识了一位被废的太妃，太妃教导她宫廷斗争的技巧。苏婉逐渐明白，要在后宫生存，必须变得强大。

终于，机会来了。皇上出巡途中遭遇刺杀，苏婉冒险救驾，重新获得了皇上的宠爱。回宫后，苏婉开始反击，她设计揭露皇后的阴谋，最终成为新的皇后。
```

### 输出

```
【阶段一：入宫（铺垫）】
- 苏婉作为礼部尚书嫡女入宫选秀
- 被皇上选中，初入后宫
- 发现后宫的复杂环境和明争暗斗

【阶段二：得宠（发展）】
- 苏婉凭借智慧和谨慎获得皇上宠爱
- 在后宫中逐渐站稳脚跟
- 引起皇后的嫉妒和敌意

【阶段三：遭陷（冲突）】
- 皇后设计陷害苏婉
- 苏婉被打入冷宫，遭遇重大挫折
- 从得宠到失宠，命运跌入谷底

【阶段四：成长（转折）】
- 在冷宫中结识被废太妃
- 学习宫廷斗争技巧，心态发生转变
- 从单纯到善于权谋，逐渐成长

【阶段五：反击（高潮与结局）】
- 皇上遇刺，苏婉冒险救驾
- 重新获得宠爱，命运出现转机
- 设计揭露皇后阴谋，成功反击
- 最终成为新的皇后
```

## 示例 3: 竖屏短剧结构 - 都市逆袭剧

### 输入

```
苏雅是一名普通的职场白领，在公司里默默无闻，经常被上司打压。某天，公司来了一位新总监顾言，对苏雅格外关注。

在重要项目中，苏雅被上司刻意刁难。顾言暗中帮助她，让她展现了自己的能力。最终，苏雅成功证明了自己，获得了晋升。

原来，顾言是苏雅大学时期的学长，一直默默关注着她。两人在工作中逐渐产生感情，最终走到一起。
```

### 输出

```
【阶段一：压抑（第1-3集）】
- 苏雅在职场中被上司刁难和打压
- 默默无闻，被同事轻视
- 新总监顾言入职，对苏雅格外关注

【阶段二：转折（第4-6集）】
- 重要项目中，上司刻意刁难苏雅
- 顾言暗中帮助苏雅
- 苏雅展现能力，获得认可
- 上司的刁难失败，但并未放弃

【阶段三：高潮（第7-8集）】
- 苏雅在关键项目中大放异彩
- 上司的阴谋被揭露
- 苏雅获得晋升，成功逆袭
- 顾言身份揭晓，是苏雅的学长

【阶段四：圆满（第9-10集）】
- 顾言向苏雅表达心意
- 两人确认关系，走到一起
- 上司被辞退，恶有恶报
```

## 示例 4: 多线叙事结构 - 悬疑推理剧

### 输入

```
刑警队长陈默接手了一起离奇的连环杀人案。凶手总是在作案现场留下一张扑克牌，每张牌的数字代表受害者的死亡顺序。

陈默调查发现，所有受害者都与十年前的一起意外事故有关。而那起事故是他父亲负责的案子，父亲在调查中意外身亡。

与此同时，法医小林在受害者身上发现了新的线索，指向一个神秘组织。陈默开始怀疑，凶手可能是十年前事故的幸存者。

最终，陈默查到真凶竟然是他最信任的搭档。搭档的家人在十年前的事故中丧生，他潜伏警队多年，就是为了复仇。在最后的对决中，陈默亲手逮捕了自己的搭档。
```

### 输出

```
【阶段一：案件发生（主线开启）】
- 连环杀人案发生，现场留下扑克牌签名
- 陈默接手案件调查
- 发现所有受害者与十年前事故有关

【阶段二：深入调查（多线并行）】
- 陈默线：发现父亲曾负责该事故调查
- 法医小林线：发现新线索，指向神秘组织
- 搭档线：暗中协助调查，但行为可疑

【阶段三：真相浮现（线索汇合）】
- 陈默发现父亲死亡真相
- 法医小林确认神秘组织存在
- 陈默开始怀疑身边的人

【阶段四：惊人反转（高潮）】
- 陈默发现真凶是搭档
- 搭坦白动机：家人在事故中丧生
- 两人展开激烈对峙

【阶段五：正义伸张（结局）】
- 陈默亲手逮捕搭档
- 案件真相大白
- 陈默完成父亲未竟的事业
```

## 示例 5: 重生复仇结构 - 重生剧

### 输入

```
林晚被丈夫顾成和闺蜜苏菲联手害死。她重生回到了三年前，刚和顾成在一起的时候。

林晚决定改变命运，她开始收集顾成和苏菲的罪证。同时，她故意接近顾成的死对头——霸道总裁傅司寒。

在公开场合，林晚揭露了顾成和苏菲的阴谋。傅司寒出现支持她，顾成和苏菲身败名裂。

最终，林晚和傅司寒在一起，获得了幸福。
```

### 输出

```
【阶段一：重生（前提设定）】
- 林晚被顾成和苏菲害死
- 重生回到三年前
- 决定改变命运，开始复仇计划

【阶段二：布局（谋划阶段）】
- 林晚表面与顾成维持关系
- 暗中收集顾成和苏菲的罪证
- 故意接近傅司寒，建立新关系

【阶段三：发展（关系推进）】
- 林晚与傅司寒关系逐渐密切
- 继续收集证据，等待时机
- 顾成和苏菲仍未察觉

【阶段四：反击（高潮爆发）】
- 在公开场合揭露阴谋
- 傅司寒出面支持林晚
- 顾成和苏菲身败名裂
- 复仇成功

【阶段五：新生（圆满结局）】
- 林晚和傅司寒确认关系
- 开始新的生活
- 顾成和苏菲受到法律制裁
```

## 情节点提炼技巧

### 1. 阶段划分原则

**三幕式结构**（适用于大多数故事）：
- 第一幕：建立（介绍人物、设置目标、建立冲突）
- 第二幕：发展（冲突升级、人物成长、情节推进）
- 第三幕：高潮与结局（最终对决、解决冲突、达成目标）

**五幕式结构**（适用于复杂故事）：
- 第一幕：铺垫（介绍背景、建立人物关系）
- 第二幕：发展（冲突出现、情节推进）
- 第三幕：冲突（危机出现、矛盾激化）
- 第四幕：转折（转机出现、局势逆转）
- 第五幕：高潮与结局（最终解决、新的平衡）

**竖屏短剧结构**（适用于短剧）：
- 压抑阶段（制造冲突、积累情绪）
- 转折阶段（出现转机、释放情绪）
- 高潮阶段（冲突爆发、情绪峰值）
- 圆满阶段（问题解决、达成目标）

### 2. 情节点表述要点

- **简洁准确**：每个情节点表述不超过150字
- **客观忠实**：严格按照原文意思总结，不自行创作
- **逻辑连贯**：确保情节点之间逻辑连贯，前后呼应
- **主次分明**：突出主要情节，略写次要情节
- **阶段清晰**：每个阶段有明确的情节主旨

### 3. 常见错误避免

- ❌ 遗漏重要情节转折点
- ❌ 阶段划分不合理，不符合故事结构
- ❌ 情节点表述过于冗长或过于简略
- ❌ 擅自添加原文不存在的情节
- ❌ 阶段之间缺乏逻辑连贯性
