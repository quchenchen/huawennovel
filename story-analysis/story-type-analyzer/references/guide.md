# 题材类型与创意提炼指南

## 一、基本定义

### 故事类型 (Story Type)
"故事类型"是一个戏剧概念，指描述和分类不同类型故事的方式或方法。它涉及到对故事元素、情节结构、主题、风格等方面进行归类和解释，以便更好地理解和分析故事。

一般以**主题**及**情节结构**来对故事进行分析：
- **主题**: 包含爱情、冒险、科幻、恐怖、喜剧、悬疑、动作、历史等。
- **情节结构**: 例如传统的三幕剧结构（起始、发展、高潮）或五幕剧结构（引子、升华、转折、高潮、结局）。

## 二、分析维度

### 1. 类型与主题
分析故事的主要叙事类型（如科幻、悬疑、爱情等）以及核心主题（如成长、复仇、救赎等）。

### 2. 情节结构
分析故事的整体架构，识别关键的情节转折点和结构模式（如英雄之旅、救猫咪等）。

### 3. 核心创意
提炼故事中最具创新性和独特性的设定或构思，这是区分于同类作品的关键。

### 4. 故事亮点
总结故事中引人入胜的元素，如独特的人物设定、意想不到的反转、深刻的情感表达等。

## 三、工作流程

1. **阅读理解**: 对提供的故事文本进行充分的阅读与理解，梳理上下文关系，解决表述重复或断裂的问题。
2. **类型定位**: 根据故事类型的定义，总结该故事主要的故事类型与叙事主题。
3. **结构分析**: 对该故事的基本情节结构进行分析与总结。
4. **创意提炼**: 总结提炼故事的核心创意。
5. **亮点总结**: 总结提炼故事的主要亮点。

## 四、输出规范

分析结果应包含以下几个部分：

- **【类型与主题】**: 总结故事文本的类型与叙事主题。
- **【情节结构】**: 总结提炼故事文本的主要情节结构。
- **【核心创意】**: 总结提炼故事文本的核心创意。
- **【故事亮点】**: 总结提炼故事文本的主要亮点。

## 五、最佳实践

- **准确性**: 严格按照故事原文所表达的内容来总结，不要自行创作。
- **精炼性**: 故事类型尽量使用词语或短语，避免冗长的句子。
- **逻辑性**: 结构分析要清晰，体现故事的发展脉络。
- **独特性**: 创意和亮点的提炼要抓住作品的独特之处，避免泛泛而谈。
