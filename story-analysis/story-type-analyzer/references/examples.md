# 题材类型与创意提炼专家 - 详细示例

## 示例 1: 奇幻穿越故事

### 输入

```
分析以下故事的题材类型：

现代青年李明在参观博物馆时，无意中触碰了一件古代文物，突然穿越到了一个武侠世界。他发现自己带去了现代的科学知识，开始用物理原理解释武功招式。他还把现代商业模式带到古代，开创了全新的商业帝国。在冒险过程中，李明结识了各路英雄好汉，最终帮助武林平息了一场浩劫。
```

### 输出

```
【题材类型分析】
- 主导题材：奇幻穿越
- 辅助题材：武侠、冒险、成长、商战
- 题材融合度：高

【创意元素提炼】
1. 核心创意：现代知识体系与古代武侠世界的碰撞融合
2. 创新点1：用现代物理原理解释和改良武功招式，带来全新的武学理解
3. 创新点2：现代商业模式在古代江湖的创新应用
4. 创新点3：文物穿越的设定，连接古今两个世界

【故事特色】
- 设定特色：古今文化碰撞产生的新鲜视角和趣味性
- 叙事特色：采用古今对比的双重视角，增强代入感

【题材价值】
- 商业价值：高，符合当下年轻观众对新鲜题材的喜好
- 艺术价值：中等，创意独特但深度有限
```

## 示例 2: 都市职场故事

### 输入

```
分析以下故事的题材类型：

28岁的林浅是广告公司的策划经理，从小城市来到北京打拼。她在工作中遇到了各种挑战：严厉的上司、难缠的客户、复杂的办公室政治。但林浅凭借自己的努力和才华，一步步在公司站稳脚跟。在经历了一次重要项目的成功后，林浅获得了晋升，实现了自己的职业梦想。
```

### 输出

```
【题材类型分析】
- 主导题材：都市职场
- 辅助题材：励志、成长、女性题材
- 题材融合度：中

【创意元素提炼】
1. 核心创意：小城市青年在大城市的奋斗与成长
2. 创新点1：真实的职场环境和挑战
3. 创新点2：女性视角的职场成长故事

【故事特色】
- 设定特色：贴近现实的职场设定，容易引发共鸣
- 叙事特色：写实主义风格，注重细节刻画

【题材价值】
- 商业价值：中，贴近现实但创新性有限
- 艺术价值：中等，能够反映现实但深度一般
```

## 示例 3: 古装宫斗故事

### 输入

```
分析以下故事的题材类型：

苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，她发现后宫中充满了勾心斗角。苏婉凭借智慧和谨慎站稳脚跟。皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉结识了被废太妃，学习了宫廷斗争技巧。皇上遇刺，苏婉冒险救驾，重新获得宠爱。苏婉设计揭露皇后阴谋，最终成为皇后。
```

### 输出

```
【题材类型分析】
- 主导题材：古装宫斗
- 辅助题材：权谋、复仇、成长
- 题材融合度：高

【创意元素提炼】
1. 核心创意：从单纯少女到深谙权谋的皇后的蜕变
2. 创新点1：冷宫成长的设定，人物转变更具说服力
3. 创新点2：从受害者到反击者的身份转变

【故事特色】
- 设定特色：复杂的宫廷关系网和权力斗争
- 叙事特色：情节紧凑，反转频出

【题材价值】
- 商业价值：高，宫斗题材有稳定的受众基础
- 艺术价值：中等，套路较为成熟但创新有限
```

## 示例 4: 悬疑推理故事

### 输入

```
分析以下故事的题材类型：

刑警队长陈默接手连环杀人案，凶手现场留扑克牌。受害者都与十年前事故有关，事故是陈默父亲负责的，父亲调查中意外身亡。陈默怀疑凶手是事故幸存者。搭档李阳对案件过于热心。陈默发现真凶是李阳。李阳家人在事故中丧生，潜伏警队复仇。对决中，陈默逮捕李阳。
```

### 输出

```
【题材类型分析】
- 主导题材：悬疑推理
- 辅助题材：犯罪、复仇、心理
- 题材融合度：高

【创意元素提炼】
1. 核心创意：最信任的人竟然是幕后黑手
2. 创新点1：扑克牌签名的设定，增加悬念
3. 创新点2：陈年旧案与当前案件的关联

【故事特色】
- 设定特色：扑克牌签名机制，增强记忆点
- 叙事特色：层层剥茧的推理过程

【题材价值】
- 商业价值：中高，悬疑推理有稳定受众
- 艺术价值：中等，反转设计巧妙但深度有限
```

## 示例 5: 甜宠爱情故事

### 输入

```
分析以下故事的题材类型：

平凡女孩苏雅在公司里默默无闻。新来的总监顾言对她格外关注。在重要项目中，上司刁难苏雅。顾言暗中帮助，苏雅展现能力，获得晋升。原来顾言是苏雅的大学学长，一直关注她。两人在工作中确认心意，走到一起。
```

### 输出

```
【题材类型分析】
- 主导题材：甜宠爱情
- 辅助题材：职场、偶像、养成
- 题材融合度：中

【创意元素提炼】
1. 核心创意：平凡女孩被霸道总裁宠爱
2. 创新点1：职场背景下的甜宠故事
3. 创新点2：学长身份的反转惊喜

【故事特色】
- 设定特色：职场与爱情的双重线索
- 叙事特色：轻松愉快的节奏，甜蜜温馨

【题材价值】
- 商业价值：高，甜宠题材有广泛的受众基础
- 艺术价值：低，以娱乐性为主
```
