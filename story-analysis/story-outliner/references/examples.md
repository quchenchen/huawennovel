# 故事大纲生成专家 - 详细示例

## 示例 1: 都市职场故事

### 输入

```
请为以下故事生成大纲：

我大学毕业来到北京，在这个陌生的城市里开始了新的生活。刚开始在公司里什么都不懂，被上司张经理骂得狗血淋头。同事小李看我可怜，教了我很多东西。慢慢地，我适应了工作，也得到了张经理的认可。下班后，我喜欢和邻居小王一起吃火锅聊天。有一天，我在街上遇到了大学时暗恋的女生小美，她也在北京工作。我们重新取得了联系，聊了很多大学时的回忆。但小美已经有男朋友了，还要准备结婚。我虽然有些失落，但还是祝福她。经过这件事，我决定更加努力工作，在北京扎根下去。
```

### 输出

```
主人公大学毕业后独自来到北京，在这个陌生的城市开始了新的生活。初到北京时，主人公对工作环境感到陌生和不安，在公司里经常因为业务不熟练而被严厉的上司张经理批评。幸运的是，同事小李看到了主人公的困难，主动提供帮助和指导。在小李的协助下，主人公逐渐适应了工作节奏，业务能力有了明显提升，最终获得了张经理的认可。在生活方面，主人公与邻居小王成为了好朋友，两人经常在下班后一起吃火锅聊天，分享彼此的生活。一次偶然的机会，主人公在街上遇到了大学时期暗恋的女生小美，发现她也在北京工作。两人重新取得联系，一起回忆大学时光的美好往事。然而主人公得知小美已经有了男朋友并且即将结婚，内心虽然有些失落，但还是真诚地祝福她。经历了这次情感上的波折后，主人公更加坚定了在北京努力工作、扎根发展的决心，对未来充满信心。
```

## 示例 2: 古装言情故事

### 输入

```
请为以下故事生成大纲：

苏婉是礼部尚书的嫡女，聪明美丽。她入宫参加选秀，被皇上选中。初入宫时，苏婉因为不熟悉宫廷规矩而犯错，被其他妃嫔嘲笑。但苏婉凭借自己的智慧和谨慎，逐渐在后宫站稳脚跟。皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉遇到了被废的太妃，太妃教导她宫廷斗争的技巧。终于，皇上遇刺，苏婉冒险救驾。皇上重新宠爱苏婉。苏婉设计揭露皇后的阴谋，最终成为皇后。
```

### 输出

```
苏婉是礼部尚书的嫡女，聪明美丽，被选入宫。初入宫廷时，因不熟悉复杂的规矩而犯错，遭到其他妃嫔的嘲笑和排挤。但苏婉凭借智慧和谨慎，逐渐适应宫廷生活，获得皇上的宠爱。皇后嫉妒苏婉得宠，设计陷害她，苏婉被打入冷宫。在冷宫中，苏婉遇到被废的太妃，太妃教导她宫廷斗争的技巧。苏婉开始改变自己，从单纯变得善于权谋。终于，皇上出巡途中遭遇刺杀，苏婉冒险救驾，重新获得皇上的宠爱。回宫后，苏婉设计揭露皇后的阴谋，皇后被打入冷宫。最终，苏婉成为新的皇后。
```

## 示例 3: 悬疑推理故事

### 输入

```
请为以下故事生成大纲：

刑警队长陈默接手了一起连环杀人案。凶手每次在现场留下一张扑克牌。陈默调查发现，所有受害者都与十年前的一起事故有关。而那起事故是陈默父亲负责的案子，父亲在调查中意外身亡。陈默开始怀疑凶手是事故的幸存者。在调查过程中，陈默的搭档李阳表现得异常热心。陈默逐渐发现，李阳对案件知道得太多。最终，陈默查到真凶竟然是李阳。李阳的家人在十年前的事故中丧生，他潜伏警队多年就是为了复仇。在对决中，陈默亲手逮捕了李阳。
```

### 输出

```
刑警队长陈默接手了一起离奇的连环杀人案，凶手每次在现场留下一张扑克牌作为标记。陈默深入调查发现，所有受害者都与十年前的一起意外事故有关联。而这起事故正是陈默父亲负责的案件，父亲在调查过程中意外身亡。陈默开始怀疑凶手可能是那起事故的幸存者，为复仇而作案。在调查过程中，搭档李阳表现得异常热心，对案件细节了如指掌。陈默逐渐察觉到李阳的异常行为，开始暗中调查自己的搭档。最终真相大白，真凶竟然就是李阳。原来李阳的家人在十年前的事故中丧生，他潜伏警队多年就是为了复仇。在最后的对决中，陈默亲手逮捕了自己的搭档，为这起连环杀人案画上句号。
```

## 示例 4: 重生复仇故事

### 输入

```
请为以下故事生成大纲：

林晚被丈夫顾成和闺蜜苏菲联手害死。她重生回到了三年前，刚和顾成在一起的时候。林晚决定改变命运，开始收集顾成和苏菲的罪证。同时，她故意接近顾成的死对头——霸道总裁傅司寒。傅司寒被林晚吸引，两人逐渐产生感情。在公开场合，林晚揭露了顾成和苏菲的阴谋。傅司寒出面支持林晚，顾成和苏菲身败名裂。最终，林晚和傅司寒在一起，获得了幸福。
```

### 输出

```
林晚被丈夫顾成和闺蜜苏菲联手陷害致死，却意外重生回到了三年前，正是她与顾成刚开始交往的时候。林晚决心改变命运，表面上继续与顾成维持恋爱关系，暗中却开始收集顾成和苏菲的罪证。同时，林晚故意接近顾成的商业对手——霸道总裁傅司寒，试图利用他的力量复仇。傅司寒被林晚的聪明和坚韧吸引，两人在频繁接触中逐渐产生了真挚的感情。在一场重要的公开商业活动上，林晚当众揭露了顾成和苏菲的阴谋诡计。傅司寒挺身而出支持林晚，提供了关键证据，使顾成和苏菲身败名裂，最终受到法律制裁。经历了背叛与重生，林晚终于找到真爱，与傅司寒携手开始新的生活。
```

## 示例 5: 职场逆袭故事

### 输入

```
请为以下故事生成大纲：

苏雅是一名普通的职场白领，在公司里默默无闻，经常被上司打压。新来的总监顾言对苏雅格外关注。在重要项目中，上司刻意刁难苏雅。顾言暗中帮助苏雅，让她展现了自己的能力。苏雅成功证明了自己，获得了晋升。原来，顾言是苏雅大学时期的学长，一直默默关注着她。两人在工作中确认了彼此的心意，走到一起。
```

### 输出

```
苏雅是一名普通的职场白领，在公司里默默无闻地工作，经常遭到上司的刻意打压和不公平对待。新任总监顾言加入公司后，对苏雅格外关注，这让苏雅感到困惑。在一次重要的项目中，上司故意刁难苏雅，试图让她出丑。关键时刻，顾言暗中帮助苏雅，让她有机会展现自己的真实能力。苏雅在项目中表现出色，成功证明了自己的价值，获得了公司高层的认可和晋升。顾言的身份终于揭晓，他原来是苏雅大学时期的学长，一直默默关注着她的成长。两人在工作中逐渐了解彼此，确认了心意，最终走到一起。
```
