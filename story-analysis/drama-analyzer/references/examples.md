# 剧本分析专家 - 详细示例

## 示例 1: 悬疑小说情节分析

### 输入

```
主角李铭是一名资深记者，某天收到一封神秘信件，声称知道其父亲二十年前意外死亡的真相。信件中提供了一个地址和时间，要求他独自前往。

李铭前往指定地点，发现是一个废弃的仓库。在仓库中，他找到了一本日记，日记的主人正是他父亲生前的同事。日记中记录了一个惊人的秘密：父亲的死并非意外，而是被人谋杀。

就在李铭准备离开时，突然遭遇不明人士跟踪。他迅速驾车离开，但在回家的路上，发现自己的车被人动了手脚，刹车失灵。
```

### 输出

```
【情节点】：主角收到神秘信件，声称知道其父亲死亡真相
【戏剧功能】：设置悬念钩子，引发观众好奇，推动主角展开调查，建立故事的核心驱动力

【情节点】：主角在废弃仓库找到日记，发现父亲死因真相
【戏剧功能】：揭示核心秘密，推动剧情发展，加深悬念，为后续冲突埋下伏笔

【情节点】：主角遭遇不明人士跟踪，车辆被人动手脚
【戏剧功能】：制造紧张氛围，升级冲突，暗示背后有秘密组织，增加紧迫感和危险感
```

## 示例 2: 爱情故事情节分析

### 输入

```
林浅是一名28岁的广告公司策划经理，独立坚强。某次项目竞标中，她遇到了竞争对手陆远。两人在竞标过程中针锋相对，但在合作过程中逐渐发现彼此的才华和魅力。

项目成功后，两人开始约会。然而，陆远的前女友突然回国，试图挽回他。面对前女友的纠缠，陆远明确拒绝，并向林浅表达了真心。最终，两人克服了所有阻碍，走到了一起。
```

### 输出

```
【情节点】：林浅和陆远在项目竞标中相遇，针锋相对
【戏剧功能】：建立初始冲突，展现人物性格，为后续关系发展埋下伏笔

【情节点】：两人在合作过程中逐渐了解彼此，感情萌芽
【戏剧功能】：化解初始冲突，推进关系发展，展现人物魅力，增强观众期待

【情节点】：前女友回国，试图挽回陆远，制造情感障碍
【戏剧功能】：设置情感冲突，考验主角感情，增加戏剧张力，推动剧情高潮

【情节点】：陆远明确拒绝前女友，向林浅表达真心
【戏剧功能】：解决情感冲突，确认主角感情，实现情感升华，满足观众期待
```

## 示例 3: 职场逆袭故事分析

### 输入

```
苏雅是一名普通的职场白领，在公司里默默无闻。某天，公司来了一位新总监，对苏雅格外关注。

在重要项目中，苏雅被上司打压。新总监暗中帮助她，让她展现了自己的能力。最终，苏雅成功证明了自己，获得了晋升。

原来，新总监是苏雅大学时期的学长，一直默默关注着她。
```

### 输出

```
【情节点】：新总监入职，对苏雅格外关注
【戏剧功能】：设置悬念钩子，暗示两人关系，引发观众好奇

【情节点】：苏雅在工作中被上司打压
【戏剧功能】：制造冲突和压抑，积累观众情绪，为后续反转蓄力

【情节点】：新总监暗中帮助苏雅
【戏剧功能】：展现人物关系，推动剧情发展，暗示真实身份

【情节点】：苏雅展现能力，成功晋升
【戏剧功能】：实现情绪释放，满足观众期待，完成人物成长

【情节点】：新总监身份揭晓，是苏雅的学长
【戏剧功能】：制造情感惊喜，深化人物关系，给观众温暖感
```

## 示例 4: 古装宫斗故事分析

### 输入

```
苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。

入宫后，皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉结识了一位被废的太妃，太妃教导她宫廷斗争的技巧。

终于，皇上出巡途中遭遇刺杀，苏婉冒险救驾，重新获得宠爱。回宫后，苏婉设计揭露皇后的阴谋，最终成为新的皇后。
```

### 输出

```
【情节点】：苏婉入宫选秀，被皇上选中
【戏剧功能】：建立故事起点，展现主角身份，为后续冲突埋下伏笔

【情节点】：皇后设计陷害，苏婉被打入冷宫
【戏剧功能】：制造重大挫折，升级冲突，推动主角成长

【情节点】：苏婉在冷宫结识太妃，学习宫廷斗争技巧
【戏剧功能】：展现主角成长，积累能力，为后续反击做准备

【情节点】：苏婉冒险救驾，重新获得宠爱
【戏剧功能】：实现命运转折，释放观众情绪，推动剧情高潮

【情节点】：苏婉揭露皇后阴谋，成为新皇后
【戏剧功能】：完成复仇弧线，实现终极目标，满足观众期待
```

## 示例 5: 重生复仇故事分析

### 输入

```
林晚被丈夫顾成和闺蜜苏菲联手害死。她重生回到了三年前，刚和顾成在一起的时候。

林晚决定改变命运，她开始收集顾成和苏菲的罪证。同时，她故意接近顾成的死对头——霸道总裁傅司寒。

在公开场合，林晚揭露了顾成和苏菲的阴谋。傅司寒出现支持她，顾成和苏菲身败名裂。最终，林晚和傅司寒在一起，获得了幸福。
```

### 输出

```
【情节点】：林晚被害，重生回到三年前
【戏剧功能】：设置核心悬念，建立故事前提，为复仇提供动机

【情节点】：林晚决定改变命运，开始收集罪证
【戏剧功能】：明确主角目标，推动剧情发展，建立复仇基调

【情节点】：林晚接近顾成的死对头傅司寒
【戏剧功能】：引入新角色，制造新的关系线，为后续反转埋伏笔

【情节点】：林晚揭露顾成和苏菲的阴谋
【戏剧功能】：实现复仇高潮，释放观众情绪，完成主要冲突

【情节点】：林晚和傅司寒在一起，获得幸福
【戏剧功能】：实现情感满足，完成故事闭环，给予观众美好结局
```
