# 故事梗概生成专家 - 详细示例

## 示例 1: 都市爱情故事

### 输入

```
请为以下故事生成梗概：

林浅是一名28岁的广告公司策划经理，独立坚强。某次项目竞标中，她遇到了竞争对手陆远。两人在竞标过程中针锋相对，但在合作过程中逐渐发现彼此的才华和魅力。项目成功后，两人开始约会。然而，陆远的前女友突然回国，试图挽回他。面对前女友的纠缠，陆远明确拒绝，并向林浅表达了真心。最终，两人克服了所有阻碍，走到了一起。
```

### 输出

```
【故事梗概】

【故事背景】
现代都市背景下，独立自强的职场女性林浅和才华横溢的创意总监陆远在同一广告行业工作。

【主要情节】
故事主要讲述林浅和陆远从竞争对手到恋人，经历各种考验最终走到一起的过程。两人在项目竞标中初识，从针锋相对到相互欣赏，感情悄然滋长。然而前女友的出现给这段感情带来了考验。

【故事发展】
故事开始时，林浅和陆远作为项目竞标中的竞争对手存在。随着项目合作的需要，两人被迫共事，在合作中发现对方的才华和魅力。项目成功后，两人开始约会，感情逐渐升温。然而，陆远的前女友突然回国，试图挽回他，制造了情感障碍。

【关键转折点】
1. 项目竞标中的针锋相对，建立初始冲突
2. 合作过程中的相互发现，感情萌芽
3. 前女友回国，制造情感障碍和考验
4. 陆远明确拒绝前女友，向林浅表达真心

【故事结局】
林浅和陆远克服了前女友的纠缠和各种阻碍，最终走到一起，在爱情和事业上都获得了圆满。
```

## 示例 2: 古装宫斗故事

### 输入

```
请为以下故事生成梗概：

苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，她发现后宫中充满了勾心斗角。苏婉凭借智慧和谨慎站稳脚跟。皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉结识了被废太妃，学习了宫廷斗争技巧。皇上遇刺，苏婉冒险救驾，重新获得宠爱。苏婉设计揭露皇后阴谋，最终成为皇后。
```

### 输出

```
【故事梗概】

【故事背景】
古代宫廷背景下，礼部尚书嫡女苏婉被选入宫，卷入复杂的后宫斗争。

【主要情节】
故事讲述苏婉从普通秀女成长为皇后的历程。苏婉初入宫时凭智慧得宠，却遭皇后嫉妒陷害。被打入冷宫后，苏婉在废太妃指导下学会宫廷斗争技巧。最终苏婉冒险救驾，重新得宠，成功反击皇后，成为六宫之主。

【故事发展】
故事开始时，苏婉入宫选秀，被皇上选中。初入后宫的苏婉凭借智慧和谨慎获得皇上宠爱，在后宫站稳脚跟。然而皇后的嫉妒引来了陷害，苏婉被打入冷宫，遭遇人生低谷。在冷宫中，苏婉结识被废太妃，学习宫廷斗争技巧，完成了从单纯到善于权谋的转变。

【关键转折点】
1. 苏婉入宫得宠，引起皇后嫉妒
2. 皇后设计陷害，苏婉被打入冷宫
3. 苏婉在冷宫中学会宫廷斗争技巧
4. 皇上遇刺，苏婉冒险救驾，命运转折
5. 苏婉揭露皇后阴谋，成功上位

【故事结局】
苏婉成功揭露皇后的阴谋，皇后被打入冷宫，苏婉成为新的皇后，完成了从受害者到胜利者的逆袭。
```

## 示例 3: 职场逆袭故事

### 输入

```
请为以下故事生成梗概：

苏雅是普通职场白领，默默无闻，经常被上司打压。新总监顾言对她格外关注。重要项目中，上司刻意刁难苏雅。顾言暗中帮助，苏雅展现能力，获得晋升。原来顾言是苏雅的大学学长，一直关注她。两人确认心意，走到一起。
```

### 输出

```
【故事梗概】

【故事背景】
现代职场背景下，普通白领苏雅在一家广告公司工作，遭遇职场不公。

【主要情节】
故事讲述苏雅从默默无闻的职场小人物，在神秘总监顾言的帮助下，通过重要项目证明自己，最终获得事业和爱情双丰收的故事。

【故事发展】
故事开始时，苏雅在公司里默默无闻，经常遭到上司的打压和刁难。新任总监顾言的加入给她的命运带来了转机。在一次重要项目中，上司刻意刁难苏雅，试图让她出丑。关键时刻，顾言暗中帮助苏雅，让她有机会展现自己的真实能力。苏雅在项目中表现出色，成功证明了自己的价值，获得了公司高层的认可和晋升。

【关键转折点】
1. 新总监顾言入职，对苏雅格外关注
2. 重要项目中，上司刻意刁难苏雅
3. 顾言暗中帮助，苏雅展现能力
4. 苏雅获得晋升，事业迎来转机
5. 顾言身份揭晓，两人确认关系

【故事结局】
苏雅成功逆袭，从默默无闻到获得晋升。顾言的身份揭晓，原来是苏雅大学时期的学长，一直默默关注她。两人在工作中确认了心意，最终走到一起。
```

## 示例 4: 重生复仇故事

### 输入

```
请为以下故事生成梗概：

林晚被丈夫顾成和闺蜜苏菲害死。她重生回到三年前，刚和顾成在一起的时候。林晚决定改变命运，开始收集顾成和苏菲的罪证。同时，她故意接近顾成的死对头——霸道总裁傅司寒。在公开场合，林晚揭露了顾成和苏菲的阴谋。傅司寒出面支持，顾成和苏菲身败名裂。林晚和傅司寒在一起，获得幸福。
```

### 输出

```
【故事梗概】

【故事背景】
现代都市背景下，林晚遭遇丈夫和闺蜜的双重背叛，意外获得重生的机会。

【主要情节】
故事讲述林晚重生后，利用前世记忆改变命运，收集罪证，最终成功复仇，并与真爱傅司寒走到一起的故事。

【故事发展】
故事开始时，林晚被丈夫顾成和闺蜜苏菲联手陷害致死，却意外重生回到了三年前。林晚决心改变命运，表面上继续与顾成维持关系，暗中开始收集两人的罪证。同时，林婉故意接近顾成的商业对手——霸道总裁傅司寒，试图利用他的力量复仇。傅司寒被林晚的聪明和坚韧吸引，两人在频繁接触中逐渐产生了真挚的感情。

【关键转折点】
1. 林晚被害，重生回到三年前
2. 林晚开始收集顾成和苏菲的罪证
3. 林晚接近傅司寒，两人产生感情
4. 公开场合揭露阴谋，复仇成功
5. 与傅司寒在一起，获得幸福

【故事结局】
在一场重要的公开商业活动上，林晚当众揭露了顾成和苏菲的阴谋诡计。傅司寒挺身而出支持林晚，提供了关键证据，使顾成和苏菲身败名裂，最终受到法律制裁。经历了背叛与重生，林晚终于找到真爱，与傅司寒携手开始新的生活。
```

## 示例 5: 悬疑推理故事

### 输入

```
请为以下故事生成梗概：

刑警队长陈默接手连环杀人案，凶手现场留扑克牌。受害者都与十年前事故有关，事故是陈默父亲负责的，父亲调查中意外身亡。陈默怀疑凶手是事故幸存者。搭档李阳对案件过于热心。陈默发现真凶是李阳。李阳家人在事故中丧生，潜伏警队复仇。对决中，陈默逮捕李阳。
```

### 输出

```
【故事梗概】

【故事背景】
现代都市背景下，刑警队长陈默接手一起连环杀人案，调查过程中发现案件与父亲意外身亡的旧案有关。

【主要情节】
故事讲述陈默在调查连环杀人案的过程中，发现真凶竟然是自己最信任的搭档李阳，最终亲手逮捕搭档的故事。

【故事发展】
故事开始时，刑警队长陈默接手了一起离奇的连环杀人案。凶手每次在现场留下一张扑克牌作为标记。陈默深入调查发现，所有受害者都与十年前的一起意外事故有关联。而这起事故正是陈默父亲负责的案件，父亲在调查过程中意外身亡。陈默开始怀疑凶手可能是那起事故的幸存者，为复仇而作案。

【关键转折点】
1. 陈默接手连环杀人案
2. 发现案件与父亲旧案有关
3. 怀疑搭档李阳行为异常
4. 发现真凶竟是搭档李阳
5. 最终对决，逮捕李阳

【故事结局】
真相大白，真凶竟然是陈默的搭档李阳。原来李阳的家人在十年前的事故中丧生，他潜伏警队多年就是为了复仇。在最后的对决中，陈默亲手逮捕了自己的搭档，为这起连环杀人案画上句号，也完成了父亲未竟的事业。
```
