# 竖屏短剧策划师 - 详细示例

## 示例 1: 策划都市逆袭短剧

### 输入

```
我有一个关于职场女性逆袭的故事创意，帮我做个短剧策划
```

### 输出

```
【竖屏短剧策划方案】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、情绪价值分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 核心情绪价值点：
   - 主价值：被轻视→打脸反击的爽感
   - 次价值：职场女性被看见、被认可的共鸣

2. 目标受众情绪需求：
   - 职场女性：渴望被认可，厌恶被轻视
   - 广大观众：喜欢看"弱者逆袭"的故事

3. 情绪触发机制：
   - 压抑点：主角被上司羞辱
   - 爆发点：主角身份曝光

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、黄金三秒钩子设计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 开篇3秒设计：
   - 视觉冲击：主角被当众羞辱
   - 情绪冲击：委屈、愤怒的表情特写

2. 情感冲击点：
   - 主角默默忍受，内心积蓄情绪
   - 观众产生同情和期待

3. 悬念设置：
   - 主角神秘电话暗示真实身份

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三、三幕式结构规划
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 第一幕：建立期待（第1-3集）
   - 身份反差：表面普通职员，实际隐藏大佬
   - 困境设置：被上司羞辱、被同事轻视

2. 第二幕：积累压抑（第4-6集）
   - 职场打压：上司故意刁难
   - 情绪积累：主角默默承受，观众情绪蓄积
   - 小反击：主角巧妙化解困难，展现实力

3. 第三幕：爆发释放（第7-10集）
   - 身份曝光：真实身份被揭露
   - 打脸时刻：上司震惊、后悔
   - 爽感释放：观众获得强烈满足

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
四、人设容器设计
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 主角人设：
   - 标签化：职场小白、隐藏大佬
   - 极致化：表面唯唯诺诺，实际能力超群
   - 反差感：平凡外表 vs 惊人身份

2. 配角人设：
   - 上司：势利眼、欺软怕硬
   - 同事：冷漠、看热闹
   - 神秘人：知晓主角身份的管家

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
五、商业化卡点设置
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 卡点1（第3集结尾）：
   主角准备反击时，接到神秘电话

2. 卡点2（第6集结尾）：
   主角身份即将曝光，关键时刻被打断

3. 卡点3（第8集结尾）：
   主角真实身份被揭露，但还有更大反转

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
六、具体策划方案
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【整体定位】
- 题材：都市职场逆袭
- 集数：10集
- 单集时长：2分钟

【核心卖点】
- 身份反差带来的爽感
- 职场压抑后的释放
- 隐藏大佬的神秘感

【制作建议】
- 强化视觉对比：主角平凡外表 vs 霸气气场
- 把握节奏：压抑-释放循环，保持观众期待
- 设置悬念：每集结尾留钩子
```
