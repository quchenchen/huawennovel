# 输出整理专家 - 详细示例

## 示例 1: 格式化为Markdown报告

### 输入

```
请将以下分析结果格式化为结构化Markdown报告：

分析结果：
故事类型：都市情感剧
主角：林浅，28岁策划经理
男主：陆远，创意总监
情节：从竞争到恋爱
结局：终成正果
```

### 输出

```
# 故事分析报告

## 一、基本信息

| 项目 | 内容 |
|------|------|
| 故事类型 | 都市情感剧 |
| 故事背景 | 职场环境 |
| 故事风格 | 轻松温馨 |

## 二、人物设定

### 主角：林浅
- **年龄**: 28岁
- **职业**: 策划经理
- **性格**: 独立自强，外冷内热

### 男主：陆远
- **职业**: 创意总监
- **性格**: 才华横溢，真诚专一

## 三、情节发展

1. **初识阶段**: 竞争对手
2. **发展阶段**: 合作相知
3. **高潮阶段**: 面对考验
4. **结局阶段**: 终成正果

## 四、故事结局

林浅和陆远克服了所有阻碍，走到了一起。

---

*报告生成时间：2026-01-11*
```

## 示例 2: 格式化为JSON输出

### 输入

```
请将以下人物信息格式化为JSON：

人物：林浅
年龄：28岁
职业：策划经理
性格：独立自强，外冷内热
```

### 输出

```json
{
  "character": {
    "name": "林浅",
    "age": 28,
    "occupation": "策划经理",
    "personality": ["独立自强", "外冷内热"],
    "background": "从小城市来到北京打拼"
  }
}
```

## 示例 3: 格式化为列表输出

### 输入

```
请将以下情节点格式化为结构化列表：

情节1：林浅和陆远在竞标中相遇
情节2：两人被迫合作
情节3：合作中产生欣赏
情节4：确立恋爱关系
情节5：前女友出现
情节6：克服阻碍
情节7：终成正果
```

### 输出

```
【情节发展列表】

阶段一：初识与竞争
1. 林浅和陆远在竞标中相遇
2. 两人作为竞争对手展开较量

阶段二：合作与相知
3. 两人被迫合作完成项目
4. 在合作中产生欣赏

阶段三：考验与圆满
5. 确立恋爱关系
6. 前女友出现，制造障碍
7. 克服阻碍，终成正果
```
