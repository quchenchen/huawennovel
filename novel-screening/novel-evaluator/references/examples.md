# 资深的故事评估专家（小说版）- 详细示例

## 示例 1: 评估都市爱情小说

### 输入

```
请对以下小说进行评估打分：

林浅和陆远在广告公司从竞争对手发展为恋人的故事。林浅是28岁的策划经理，独立自强。陆远是创意总监，才华横溢。两人在竞标中初识，在合作中相知。前女友的出现制造了考验，但两人最终克服阻碍走到一起。
```

### 输出

```
【小说评估报告】

【市场潜力】：评分：7.8分
- 受众适合度：高度贴合都市情感类目标受众，尤其适合25-35岁女性观众
- 讨论热度：题材贴近生活，容易引发观众对初恋、重逢的共鸣
- 稀缺性：破镜重圆题材较为常见，创新性一般
- 播放数据：符合当下都市情感剧的市场需求，预期表现良好

【创新属性】：评分：7.5分
- 核心选点：职场爱情，但同类题材较多
- 故事概念：情感真挚，但故事概念相对传统
- 故事设计：叙事结构较为完整，人物成长线清晰

【内容亮点】：评分：7.8分
- 主题立意：探讨爱情与成长，有一定深度
- 故事情境：情感细腻，但戏剧张力一般
- 人物设定：人物形象立体，但创新性不足
- 人物关系：情感关系刻画细腻，有感染力
- 情节桥段：情节发展较为平缓，缺少爆点

【总体评价】：评分：7.7分
该IP整体质量中等偏上，情感真挚细腻，人物关系刻画到位，但创新性和戏剧性相对不足。相比同类都市情感IP，竞争力中等偏上。

【跟进建议】：建议推进，但需要进行以下修改：
1. 增强情节的戏剧张力和冲突强度
2. 在人物设定上增加更多创新元素
3. 优化节奏，增加记忆点和爆点
```

## 示例 2: 评估古言小说

### 输入

```
请对以下小说进行评估打分：

苏婉从礼部尚书嫡女到皇后的历程。入宫后得宠，皇后嫉妒陷害。苏婉被打入冷宫，结识废太妃。太妃教导苏婉宫廷斗争技巧。最终苏婉揭露皇后阴谋，成为皇后。
```

### 输出

```
【小说评估报告】

【市场潜力】：评分：8.2分
- 受众适合度：精准贴合古装宫斗类目标受众
- 讨论热度：宫斗题材一直有稳定的受众基础
- 稀缺性：题材成熟但套路化严重
- 播放数据：市场需求稳定但竞争激烈

【创新属性】：评分：7.0分
- 核心选点：经典的宫斗复仇，创新性有限
- 故事概念：故事概念传统，缺少独特记忆点
- 故事设计：叙事结构完整，但套路化严重

【内容亮点】：评分：8.0分
- 主题立意：探讨权力与人性的主题
- 故事情境：宫廷斗争环境刻画到位
- 人物设定：主角成长弧线完整
- 人物关系：复杂多样，层次丰富
- 情节桥段：情节设计精妙，张弛有度

【总体评价】：评分：7.7分
该IP属于成熟的宫斗题材，整体质量中上。故事结构完整，人物刻画细致，但创新性不足。在当前竞争激烈的市场环境下，需要做出差异化才能脱颖而出。

【跟进建议】：谨慎推进，建议进行大幅度的优化后再考虑开发：
1. 在经典框架下增加创新元素
2. 丰富配角人物，增强层次
3. 在细节上做精做细
4. 寻找差异化定位
```

## 示例 3: 评估重生复仇小说

### 输入

```
请对以下小说进行评估打分：

林晚被丈夫顾成和闺蜜苏菲害死。她重生回到三年前，开始收集罪证。同时接近顾成的死对头傅司寒。最终揭露阴谋，顾成和苏菲受罚。林晚和傅司寒走到一起。
```

### 输出

```
【小说评估报告】

【市场潜力】：评分：8.5分
- 受众适合度：精准贴合重生复仇类目标受众
- 讨论热度：重生题材热度高，容易引发讨论
- 稀缺性：重生+复仇组合较为常见
- 播放数据：符合当下市场需求

【创新属性】：评分：7.5分
- 核心选点：重生复仇是最热门的题材之一
- 故事概念：故事概念清晰但套路化
- 故事设计：叙事结构完整，节奏紧凑

【内容亮点】：评分：8.3分
- 主题立意：探讨正义与复仇的主题
- 故事情境：戏剧张力十足，爽感强烈
- 人物设定：主角成长弧线完整
- 人物关系：复杂多样，关系转变精彩
- 情节桥段：情节节奏紧凑，爽感十足

【总体评价】：评分：8.1分
该IP属于热门题材，整体质量良好，复仇线设计到位，爽感十足。虽然创新性有限，但执行力较强，能够满足目标受众的需求。

【跟进建议】：建议推进，同时进行优化：
1. 在经典框架下做些创新尝试
2. 丰富配角人物层次
3. 增加更多惊喜和反转
4. 保持爽感的同时增加深度
```
