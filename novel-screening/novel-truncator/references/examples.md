# 文本截断工具（小说版）- 详细示例

## 示例 1: 在句号处截断

### 输入

```
小说文本="这是一个示例的小说文本内容。这是第二句话。这是第三句话。", 最大长度=20
```

### 输出

```
【文本截断结果】

原始长度：32
截断后长度：10
截断位置：在第一句句号处

截断后的文本：
这是一个示例的小说文本内容。
```

## 示例 2: 在换行处截断

### 输入

```
小说文本="第一章开始

第二章继续

第三章发展", 最大长度=30
```

### 输出

```
【文本截断结果】

原始长度：29
截断后长度：15
截断位置：在第一章结束换行处

截断后的文本：
第一章开始

第二章继续
```

## 示例 3: 截断较长的小说段落

### 输入

```
小说文本="这是一段很长的小说文本内容，需要进行智能截断处理。它包含了多个句子，每个句子都有其完整的意思。我们需要在最合适的位置进行截断，以确保截断后的文本保持语义的完整性和连贯性。", 最大长度=50
```

### 输出

```
【文本截断结果】

原始长度：107
截断后长度：50
截断位置：在第二句句号处

截断后的文本：
这是一段很长的小说文本内容，需要进行智能截断处理。它包含了多个句子，每个句子都有其完整的意思。
```

## 示例 4: 带截断标记

### 输入

```
小说文本="这是一段很长的小说文本内容，需要被截断。这是第二段内容，也需要被截断。这是第三段内容，同样需要被截断。", 最大长度=60, 截断标记="...[未完待续]"
```

### 输出

```
【文本截断结果】

原始长度：67
截断后长度：72
截断位置：在第二段句号处

截断后的文本：
这是一段很长的小说文本内容，需要被截断。这是第二段内容，也需要被截断。...[未完待续]
```

## 示例 5: 智能截断保持语义

### 输入

```
小说文本="故事讲述了一个年轻女孩从乡村来到城市的成长历程。她经历了许多困难和挫折，但从未放弃自己的梦想。最终，她通过自己的努力和坚持，实现了人生的价值和目标。这个故事告诉我们，只要坚持不懈，就一定能够成功。", 最大长度=80, 智能截断=true
```

### 输出

```
【文本截断结果】

原始长度：102
截断后长度：81
截断位置：在第三句句号处

截断后的文本：
故事讲述了一个年轻女孩从乡村来到城市的成长历程。她经历了许多困难和挫折，但从未放弃自己的梦想。最终，她通过自己的努力和坚持，实现了人生的价值和目标。
```
