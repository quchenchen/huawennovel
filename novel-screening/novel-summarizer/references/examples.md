# 故事大纲生成专家（小说版）- 详细示例

## 示例 1: 都市情感小说

### 输入

```
请为以下小说生成500-800字的故事大纲：

林浅是一名28岁的广告公司策划经理，从小城市来到北京打拼。在竞标项目中，她遇到了竞争对手陆远。两人在竞标过程中针锋相对，但在合作过程中逐渐发现彼此的才华和魅力。项目成功后，两人开始约会。然而，陆远的前女友突然回国，试图挽回他。面对前女友的纠缠和家庭背景差异的考验，两人最终克服了所有阻碍，走到了一起。
```

### 输出

```
主人公林浅是一名28岁的广告公司策划经理，从小城市独自来到北京打拼。在这个陌生的城市里，她凭借自己的努力和才华，在公司站稳脚跟。在一次重要的项目竞标中，林浅遇到了竞争对手陆远。陆远是公司的创意总监，才华横溢但性格内向。两人在竞标过程中针锋相对，各不相让，都想证明自己的方案更优秀。然而，公司高层决定两人必须合作完成这个重要项目。在被迫的合作过程中，林浅和陆远逐渐发现彼此的优点和魅力。林浅欣赏陆远的才华和创意，陆远则被林浅的坚韧和独立所吸引。随着项目的深入，两人之间的竞争逐渐转变为欣赏和默契。项目成功后，公司举办了庆祝晚宴。在晚宴上，陆远主动邀请林浅跳舞，两人在舞池中相谈甚欢，彼此的距离进一步拉近。从那天晚上开始，林浅和陆远开始了正式的约会。然而，他们的感情并非一帆风顺。陆远的前女友突然从国外回来，试图挽回这段感情。前女友的介入让林浅和陆远的关系出现了裂痕，两人产生了误会和隔阂。与此同时，林浅的父母得知她和一个富二代交往，担心她受到伤害，也表达了反对意见。面对前女友的纠缠和家庭背景差异带来的压力，林浅和陆远都陷入了痛苦和困惑。在经历了一段时间的分离和思考后，林浅和陆远都意识到彼此的重要性。陆远明确拒绝前女友，向她表明自己已经放下了过去。林浅也和父母进行了深入的沟通，让他们看到了陆远的真诚和决心。最终，林浅和陆远克服了所有阻碍，重新走到了一起。他们的感情经过了考验，变得更加成熟和稳固。在朋友和家人的祝福下，两人携手开始了新的生活。
```

## 示例 2: 古装宫斗小说

### 输入

```
请为以下小说生成500-800字的故事大纲：

苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，苏婉发现后宫中充满了勾心斗角。苏婉凭借智慧和谨慎，在后宫站稳脚跟。皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉结识了被废的太妃，太妃教导她宫廷斗争技巧。皇上出巡遇刺，苏婉冒险救驾，重新获得宠爱。苏婉设计揭露皇后的阴谋，最终成为皇后。
```

### 输出

```
苏婉是礼部尚书的嫡女，自幼聪慧过人，容貌清丽。在皇宫选秀的年份，苏婉被选中入宫。初入宫廷的她并不适应这个复杂的环境，但她凭借着聪慧的头脑和谨慎的性格，逐渐在后宫中站稳了脚跟。皇上的宠爱让苏婉成为了后宫中的焦点，但也引来了皇后和其他妃嫔的嫉妒。皇后是皇上在潜邸时的发妻，地位尊贵，但多年无子让她倍感威胁。她视苏婉为眼中钉，设计陷害苏婉在皇上面前失仪。苏婉被打入冷宫，从得宠的妃嫔变成了被遗忘的废人。在冷宫的日子里，苏婉心灰意冷，但一次偶然的机会，她结识了被废的太妃。太妃曾经也是后宫的宠妃，因被皇后陷害而失去一切。太妃看到苏婉的遭遇，对她心生怜悯，决定教导她宫廷斗争的技巧。在太妃的指导下，苏婉开始改变自己，从单纯的少女逐渐变得深不可测。她学会了隐忍和等待，学会了在后宫的复杂环境中保护自己。终于，机会来了。皇上出巡途中遭遇刺客袭击，随行的侍卫死伤惨重。苏婉冒险救驾，用自己的身体为皇上挡下了致命一击。她的忠诚和勇敢感动了皇上，也让她重新获得了皇上的宠爱。回到宫中后，苏婉不再像以前那样单纯。她开始在暗中收集皇后陷害她的证据，同时积极拉拢其他妃嫔，建立自己的势力。经过周密的布局，苏婉在朝廷上公开揭露了皇后的阴谋。证据确凿，皇后被打入冷宫，苏婉成功上位，成为了新的皇后。站在权力的巅峰，苏婉回首过往，心中感慨万千。她从一个单纯的少女变成了深谙权谋的皇后，这一路走来，既有自己的努力，也有太多人的牺牲。她决定做一个好皇后，用自己的权力为后宫带来公正和秩序。
```

## 示例 3: 职场励志小说

### 输入

```
请为以下小说生成500-800字的故事大纲：

苏雅是一名普通的职场白领，在公司里默默无闻，经常被上司打压。新来的总监顾言对苏雅格外关注。在重要项目中，上司刻意刁难苏雅。顾言暗中帮助苏雅，让她展现了自己的能力。苏雅成功证明了自己，获得了晋升。原来顾言是苏雅的大学学长，一直默默关注着她。两人在工作中确认了彼此的心意，走到一起。
```

### 输出

```
苏雅是一名普通的职场白领，在一家广告公司工作了三年，却一直默默无闻。她的上司是一个势利眼的小人，经常刻意刁难和打压苏雅，让她在公司里处于被忽视的位置。新任总监顾言的加入给公司带来了新的气象。顾言年轻有为，是业内知名的创意天才。让苏雅惊讶的是，这位新总监对她格外关注，经常在工作上给予她指导和帮助。在一次重要的项目中，上司为了让自己的亲信出彩，故意将苏雅排除在核心团队之外。顾言看在眼里，他没有直接干预，而是暗中创造机会让苏雅展现自己的能力。在一次客户会议上，苏雅抓住机会，提出了一个精彩的创意方案，赢得了客户的高度赞赏。这次成功让公司高层看到了苏雅的才华，她终于获得了应有的认可和机会。在随后的工作中，苏雅凭借自己的努力和顾言的支持，在公司里逐渐站稳了脚跟，最终获得了晋升，从普通白领成为了项目主管。在一次公司聚餐中，顾言向苏雅透露了一个秘密：原来他是苏雅大学时期的学长，一直默默关注着她的成长。大学时期，顾言就对这个努力学习的学妹印象深刻，但没有勇气表达自己的心意。毕业后，顾言一直在寻找机会接近苏雅，直到这次调职来到她的公司。得知真相的苏雅感动不已，她回想起和顾言相处的点点滴滴，终于明白了他对自己的特别关照。两人在工作中确认了彼此的心意，走到了一起。苏雅从默默无闻到职场成功的逆袭，不仅实现了自己的职业梦想，也收获了真挚的爱情。这个故事告诉我们，只要坚持不懈，努力奋斗，就一定能够迎来属于自己的春天。
```
