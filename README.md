## Key Rules Applied (v2)
- Short novel format: ≤30,000 words, 17 chapters
- Bond body sensation: specific, every chapter
- Sera: cold-calm, has-a-knife, active choices
- Kael: double-standard protector, "he suffers more"
- Cassius: elegant + cold crack underneath
- Internal monologue ≤ 15% per chapter
- Every chapter ends with a hook
