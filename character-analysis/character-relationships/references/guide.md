# 人物关系分析指南

## 一、核心定义

### 人物关系 (Character Relationship)
人物关系是指在故事或剧本中，各个角色之间的互动、联系和相互影响。

这些关系包罗万象，包括：
- **家庭关系**: 父母、子女、兄弟姐妹、夫妻等。
- **社会关系**: 朋友、同学、同事、邻居等。
- **情感关系**: 恋人、暗恋、前任、情敌等。
- **功能关系**: 师徒、主仆、盟友、敌人、竞争对手等。

人物关系对于故事的发展起着至关重要的作用，它们通常帮助塑造故事的冲突和解决方案，并推动故事进程。通过分析人物关系，可以更深入地理解故事的主题和含义。

## 二、分析维度

### 1. 关系类型
明确界定人物之间的关系属性。
- **强关系**: 血缘、婚姻、生死之交。
- **弱关系**: 点头之交、临时合作。
- **正向关系**: 互助、喜爱、支持。
- **负向关系**: 仇恨、嫉妒、阻碍。

### 2. 关系特点
描述该段关系的独特之处。例如：
- "相爱相杀"的兄弟关系。
- "亦师亦友"的上下级关系。
- "表面和谐实则暗流涌动"的夫妻关系。

### 3. 发展过程
分析关系随时间的变化轨迹：
- **建立**: 关系如何开始？
- **发展**: 经历了哪些关键事件加深或疏远？
- **转折**: 有无重大变故导致关系性质改变？
- **结局**: 故事结束时关系处于什么状态？

### 4. 关系影响
分析该关系对故事的作用：
- 如何推动情节发展？
- 如何塑造人物性格？
- 如何表达故事主题？

## 三、生成规范

1. **数量要求**: 必须分析不少于12对人物关系（如果故事体量允许），确保覆盖主要和次要人物网络。
2. **内容准确**: 严格按照故事文本原文总结，不进行主观臆测。
3. **全面细致**: 不仅要定性关系，还要分析互动细节和变化原因。

## 四、工作流程

1. **人物识别**: 罗列故事中的所有关键人物。
2. **关系连线**: 构建人物关系矩阵，识别两两之间的联系。
3. **深度分析**: 对每一对关系进行"定性"（类型）和"定量"（互动深度）分析。
4. **总结撰写**: 按照【人物A-人物B】的格式，撰写详细的关系介绍。
