# 人物小传生成专家 - 详细示例

## 示例 1: 职场女性主角

### 输入

```
请为以下故事中的主角生成人物小传：

林浅是一名28岁的广告公司策划经理，从小城市来到北京打拼。她在工作中遇到了各种挑战，但凭借自己的努力和才华，一步步在公司站稳脚跟。在竞标项目中，她遇到了竞争对手陆远。两人从针锋相对到相互欣赏，最终走到一起。
```

### 输出

```
【人物小传：林浅】

一、基本信息
- 姓名：林浅
- 年龄：28岁
- 身份：广告公司策划经理
- 外貌特征：身材高挑，气质干练，眼神坚定

二、性格特点
- 主要性格：独立自强，坚韧不拔，外冷内热
- 行为特征：做事果断，追求完美，不轻易示弱
- 语言特点：言简意赅，直击要害，有时略显尖锐

三、背景故事
- 家庭背景：小城市普通家庭，父母是教师
- 成长经历：从小成绩优异，靠自己努力考上北京名牌大学
- 重要经历：大学毕业后独自闯荡北京，从实习生做到策划经理

四、人物关系
- 与陆远：从竞争对手发展为恋人
- 与小李：同事兼好友，相互支持
- 与张经理：上下级关系，亦师亦友

五、在故事中的作用
- 功能定位：故事的核心推动者
- 情节推动：通过她的选择和行动推动情节发展
- 主题表达：展现现代女性独立自强的精神
```

## 示例 2: 古装宫廷女主角

### 输入

```
请为以下故事中的主角生成人物小传：

苏婉是礼部尚书的嫡女，聪慧过人。她入宫选秀，被皇上选中。入宫后，苏婉凭借智慧和谨慎站稳脚跟。皇后嫉妒苏婉得宠，设计陷害她。苏婉被打入冷宫。在冷宫中，苏婉结识了被废太妃，学习了宫廷斗争技巧。最终，苏婉揭露皇后阴谋，成为皇后。
```

### 输出

```
【人物小传：苏婉】

一、基本信息
- 姓名：苏婉
- 年龄：18岁（入宫时）→ 22岁（成为皇后时）
- 身份：礼部尚书嫡女 → 秀女 → 妃嫔 → 皇后
- 外貌特征：容貌清丽，气质温婉，眼神聪慧

二、性格特点
- 主要性格：聪慧过人，谨慎隐忍，善于学习
- 行为特征：行事低调，善于观察，从不多言
- 语言特点：轻声细语，言辞得体，善于聆听

三、背景故事
- 家庭背景：礼部尚书嫡女，从小接受良好教育
- 成长经历：精通琴棋书画，聪慧过人，备受父亲喜爱
- 重要经历：入宫选秀被选中，从得宠到被打入冷宫，再到最终成为皇后

四、人物关系
- 与皇上：君臣关系，后成为夫妻
- 与皇后：敌对关系，从嫉妒到陷害再到被反击
- 与废太妃：师徒关系，太妃教导她宫廷斗争技巧

五、在故事中的作用
- 功能定位：故事的核心主角
- 情节推动：通过她的成长和转变推动情节发展
- 主题表达：展现从单纯到复杂的成长历程
```

## 示例 3: 职场逆袭主角

### 输入

```
请为以下故事中的主角生成人物小传：

苏雅是一名普通的职场白领，在公司里默默无闻，经常被上司打压。新来的总监顾言对苏雅格外关注。在重要项目中，上司刻意刁难苏雅。顾言暗中帮助苏雅，让她展现了自己的能力。苏雅成功证明了自己，获得了晋升。原来顾言是苏雅的大学学长，一直默默关注着她。两人在工作中确认了彼此的心意，走到一起。
```

### 输出

```
【人物小传：苏雅】

一、基本信息
- 姓名：苏雅
- 年龄：26岁
- 身份：普通职场白领 → 项目经理
- 外貌特征：相貌平凡但干净整洁，眼神温和

二、性格特点
- 主要性格：内向害羞，勤奋踏实，不善言辞
- 行为特征：默默工作，不争不抢，任劳任怨
- 语言特点：说话轻声细语，不善表达，但言之有物

三、背景故事
- 家庭背景：普通工薪家庭，父母是普通职员
- 成长经历：从小性格内向，努力学习，成绩中等
- 重要经历：大学毕业后进入广告公司，一直默默无闻地工作

四、人物关系
- 与顾言：学长学妹关系，后发展为恋人
- 与上司：被压制与反压制的关系
- 与同事：友好但疏远的同事关系

五、在故事中的作用
- 功能定位：故事的逆袭主角
- 情节推动：通过她的成长和逆袭推动情节发展
- 主题表达：展现平凡人的逆袭可能
```
