# 人物小传生成指南

## 一、核心定义

### 重要人物 (Important Character)
"重要人物"是一个戏剧概念，也被称为"主角"或者"重要角色"，是推动故事发展和情节进展的关键人物。

他们的行为、决定以及与其他人物的互动通常构成了故事的核心部分。主要人物不一定都是正面的英雄角色，他们可能是反派，甚至是复杂性格的配角。但无论如何，他们在故事中都占据着重要的地位，并对故事的结果产生直接影响。

## 二、分析维度

### 1. 基本信息
- **姓名**: 角色的全名或常用称呼。
- **身份**: 职业、社会地位、家庭角色等。
- **外貌**: 体貌特征、穿着打扮、气质神态。

### 2. 性格特征
- **主要性格**: 核心性格关键词（如勇敢、懦弱、精明、憨厚）。
- **行为模式**: 在压力或冲突下的习惯性反应。
- **语言风格**: 说话的方式、口头禅、语速语调。

### 3. 背景故事
- **成长经历**: 塑造当前性格的关键过往事件。
- **家庭背景**: 原生家庭的影响。
- **重要经历**: 故事开始前的关键转折点。

### 4. 关系与功能
- **人物关系**: 与其他主要角色的互动模式。
- **故事功能**: 在剧情中承担的作用（如阻碍者、帮助者、导师）。
- **目标与困境**: 角色想要什么（欲望）以及面临什么障碍（冲突）。

## 三、生成规范

1. **数量要求**: 必须找出故事文本中所有的主要人物与重要配角，通常不少于8个（如果故事人物足够多）。
2. **字数控制**: 每个人物小传严格控制在300-500字之间。
3. **内容来源**: 严格基于故事文本原文分析，不进行无中生有的创作。
4. **叙事视角**: 即使原文是第一人称，小传也应使用第三人称客观描述。

## 四、工作流程

1. **深度阅读**: 通读全文，标记所有出场人物及其行为。
2. **人物筛选**: 甄别推动情节发展的"重要人物"。
3. **信息整合**: 将散落在文中的人物碎片信息（外貌、对话、行为）拼接完整。
4. **小传撰写**: 按照结构化格式撰写流畅的人物小传，确保包含性格、背景、关系、目标等核心要素。
