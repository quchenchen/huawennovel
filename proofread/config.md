# 审校配置

> 最后更新: 2026-05-06

---

## 默认设置

### 审校范围
- 默认执行全部三层检查（错字语病 + 信息核查 + 政治宗教）
- 用户可指定单项

### 严格度（按稿件类型自动调整）

| 类型 | 错字语病 | 信息核查 | 政治宗教 | 格式 |
|------|---------|---------|---------|------|
| 新闻稿 | 严格 | 严格 | 严格 | 严格 |
| 公文/红头文件 | 严格 | 严格 | 严格 | 最严 |
| 学术论文 | 严格 | 严格 | 中等 | 严格 |
| 公众号/自媒体 | 中等 | 中等 | 严格 | 宽松 |
| 博客/随笔 | 中等 | 宽松 | 中等 | 宽松 |
| 小说/文学 | 宽松 | 宽松 | 中等 | 宽松 |

### 信息核查策略
- 默认：AI 知识库验证 + 标注存疑项
- 可选：联网搜索（需用户授权）

### 修改模式
- 默认：生成审校报告，不自动修改原稿
- 可选：直接修改 / 原文批注 / 仅处理高风险

### 报告存档
- 路径：proofread/reports/
- 格式：YYYY-MM-DD-[稿件标题].md
- 保留策略：永久保留

### 版本管理
- 是否启用 Git：否（默认）
- 分支策略：main（如启用）
