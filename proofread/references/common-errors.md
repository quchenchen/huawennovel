## 一、高频错别字

### 1.1 形近字

| ❌ 错误 | ✅ 正确 | 说明 |
|---------|--------|------|
| 帐号 | 账号 | "账"用于金钱相关 |
| 帐户 | 账户 | 同上 |
| 帐目 | 账目 | 同上 |
| 象样 | 像样 | 表示相似用"像" |
| 象是 | 像是 | 同上 |
| 决对 | 绝对 | "绝"表示绝对 |
| 常年（部分场景） | 长年 | "长年"指一年到头，"常年"指长期 |
| 渡过难关 | 度过难关 | "度"用于时间，"渡"用于空间 |
| 提纲挈领 | 提纲契领 | "挈"非"契" |
| 鼎立相助 | 鼎力相助 | "鼎力"非"鼎立" |
| 蛛丝蚂迹 | 蛛丝马迹 | "马迹"非"蚂迹" |
| 走头无路 | 走投无路 | "投"非"头" |
| 名符其实 | 名副其实 | "副"非"符" |
| 按步就班 | 按部就班 | "部"非"步" |
| 一愁莫展 | 一筹莫展 | "筹"非"愁" |
| 川流不息 | 穿流不息 | "川"非"穿" |
| 墨守陈规 | 墨守成规 | "成"非"陈" |
| 迫不急待 | 迫不及待 | "及"非"急" |
| 金碧辉煌 | 金壁辉煌 | "碧"非"壁" |
| 再接再励 | 再接再厉 | "厉"非"励" |
| 变本加励 | 变本加厉 | "厉"非"励" |
| 世外桃园 | 世外桃源 | "源"非"园" |
| 脍灸人口 | 脍炙人口 | "炙"非"灸" |
| 饮鸠止渴 | 饮鸩止渴 | "鸩"非"鸠" |
| 矫揉造作 | 娇揉造作 | "矫"非"娇" |

### 1.2 音近字

| ❌ 错误 | ✅ 正确 | 说明 |
|---------|--------|------|
| 必须/必需混淆 | 必须（动词前）/ 必需（名词前） | 用法区分 |
| 反映/反应混淆 | 反映（汇报）/ 反应（回应） | 用法区分 |
| 权利/权力混淆 | 权利（公民）/ 权力（职权） | 用法区分 |
| 制定/制订混淆 | 制定（确定）/ 制订（研究起草） | 用法区分 |
| 融化/溶化混淆 | 融化（冰雪）/ 溶化（固体入液体） | 用法区分 |
| 启事/启示混淆 | 启事（公告）/ 启示（启发） | 用法区分 |
| 截止/截至混淆 | 截止（不再进行）/ 截至（到某个时间） | 用法区分 |
| 蒸气/蒸汽混淆 | 蒸气（气体）/ 蒸汽（水蒸气） | 用法区分 |
| 学历/学力混淆 | 学历（学习经历）/ 学力（学习能力） | 用法区分 |

### 1.3 的/地/得

| ❌ 错误 | ✅ 正确 | 说明 |
|---------|--------|------|
| 认真的学习 | 认真地学习 | 修饰动词用"地" |
| 跑的很快 | 跑得很快 | 补语用"得" |
| 我的书 | 我的书（正确） | 修饰名词用"的" |
| 高兴的说 | 高兴地说 | 修饰动词用"地" |
| 做的好 | 做得好 | 补语用"得" |

### 1.4 做/作

| ❌ 错误 | ✅ 正确 | 说明 |
|---------|--------|------|
| 做为 | 作为 | "作为"是介词/动词 |
| 作贡献 | 做贡献 | "做"接具体事物 |
| 做文章 | 作文章 | 抽象事物用"作" |
| 做报告 | 作报告 | 抽象事物用"作" |
| 做研究 | 做研究 | 此处"做"正确 |

## 二、高频语法错误

### 2.1 成分残缺

| 错误类型 | ❌ 错误示例 | ✅ 正确示例 |
|---------|-----------|-----------|
| 缺主语 | 通过这次活动，使大家深受教育。 | 这次活动使大家深受教育。 |
| 缺主语 | 由于他的努力，使项目顺利完成。 | 由于他的努力，项目顺利完成。 |
| 缺谓语 | 我们要建立和健全各项规章制度。 | 我们要建立健全各项规章制度。 |
| 缺宾语 | 他提出了一个很好的建议。 | 他提出了一个很好的建议。（此句正确，需具体判断） |

### 2.2 搭配不当

| 错误类型 | ❌ 错误示例 | ✅ 正确示例 |
|---------|-----------|-----------|
| 动宾不当 | 提高工作效率和质量。 | 提高工作效率，保证工作质量。 |
| 主谓不当 | 他的写作水平明显改进了。 | 他的写作水平明显提高了。 |
| 修饰语不当 | 他是多少个死难者中幸免的一个。 | 他是死难者中幸免的一个。 |
| 动补不当 | 他把房间打扫得干干净净。（正确） | — |

### 2.3 语序不当

| ❌ 错误示例 | ✅ 正确示例 |
|-----------|-----------|
| 我们应该发挥广大青年的充分作用。 | 我们应该充分发挥广大青年的作用。 |
| 他把这个问题不放在心上。 | 他不把这个问题放在心上。 |

### 2.4 句式杂糅

| ❌ 错误示例 | ✅ 正确示例（任选一） |
|-----------|-----------|
| 原因是因为…… | 原因是…… / 是因为…… |
| 靠的是……取得的 | 靠的是…… / 是……取得的 |
| 围绕以……为中心 | 围绕……中心 / 以……为中心 |
| 本着……为原则 | 本着……原则 / 以……为原则 |
| 对于……问题上 | 对于……问题 / 在……问题上 |
| 是为了……为目的 | 是为了…… / 以……为目的 |
| 分为……两种 | 分为……两种（正确）/ 有……两种 |

### 2.5 重复赘余

| ❌ 错误示例 | ✅ 正确示例 |
|-----------|-----------|
| 大约……左右 | 大约…… / ……左右 |
| 亲眼目睹 | 亲眼看到 / 目睹 |
| 非常罕见 | 极其罕见 |
| 悬而未决至今 | 悬而未决 |
| 凯旋归来 | 凯旋（"旋"已有归来之意） |
| 过于溺爱 | 溺爱（"溺"已有过度之意） |
| 涉及到 | 涉及 / 涉及到（现代汉语已可接受） |
| 被人所…… | 被…… / 为人所…… |
| 忍俊不禁地笑 | 忍俊不禁（已有笑意） |

### 2.6 否定不当

| ❌ 错误示例 | ✅ 正确示例 |
|-----------|-----------|
| 防止不再发生 | 防止再次发生 |
| 难道能否认不是这样吗 | 难道能否认是这样吗 |
| 为了防止不再发生类似事故 | 为了防止再次发生类似事故 |
| 避免不再出错 | 避免出错 |

### 2.7 关联词搭配错误

| ❌ 错误示例 | ✅ 正确示例 |
|-----------|-----------|
| 不但……而且……（主语位置错误） | 主语在关联词前：他不但会唱歌，而且会跳舞。 |
| 无论……和…… | 无论……还是…… |
| 只有……就…… | 只有……才…… |
| 只要……才…… | 只要……就…… |

## 三、标点常见错误

| ❌ 错误 | ✅ 正确 | 说明 |
|---------|--------|------|
| ... | …… | 中文省略号六个点 |
| -- | —— | 中文破折号两个 |
| "..." | "……" | 中文使用直角引号 |
| (...) | （……） | 中文使用全角括号 |
| ，连续不断 | 适当断句 | 避免一逗到底 |
| ！过多 | 适度使用 | 正式稿件慎用叹号 |
| ：用于提示语后 | 正确使用 | 冒号使用规范 |
