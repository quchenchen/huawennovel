# IP评估专家 - 详细示例

## 示例：评估网络小说改编潜力

### 输入

```
请评估以下网络小说的改编潜力：

小说名：《时光里的我们》
作者：张三
类型：都市情感

简介：讲述一对大学恋人因误会分开，十年后重逢破镜重圆的故事

市场表现：
- 网络点击量：5000万
- 读者评分：8.2分
- 粉丝基础：良好

作者信息：
- 张三，已出版三部小说
- 过往作品口碑中等偏上
```

### 输出

```
【IP评估报告】

一、基本信息
- 故事梗概：讲述一对大学恋人因误会分开，十年后重逢破镜重圆的故事
- 故事主题：爱情、成长、时间与记忆
- 人物关系：男女主从恋人到陌路人，最终重新走到一起
- 市场表现：网络点击量5000万，评分8.2分，粉丝基础良好
- 作者信息：张三，已出版三部小说，口碑中等偏上

二、市场潜力分析
- 受众适合度：高度贴合都市情感类目标受众，尤其适合25-35岁女性观众
  评分：8.2分
- 讨论热度：题材贴近生活，容易引发观众对初恋、重逢的共鸣
  评分：7.8分
- 稀缺性：破镜重圆题材较为常见，创新性一般
  评分：7.0分
- 播放数据：符合当下都市情感剧的市场需求，预期表现良好
  评分：8.0分

三、创新属性分析
- 核心选点：时间跨度十年的重逢设定，但同类题材较多
  评分：7.5分
- 故事概念：情感真挚，但故事概念相对传统
  评分：7.2分
- 故事设计：叙事结构较为完整，人物成长线清晰
  评分：7.8分

四、内容亮点分析
- 主题立意：探讨时间与爱情的永恒主题，有一定深度
  评分：7.5分
- 故事情境：情感细腻，但戏剧张力一般
  评分：7.0分
- 人物设定：人物形象立体，但创新性不足
  评分：7.3分
- 人物关系：情感关系刻画细腻，有感染力
  评分：8.0分
- 情节桥段：情节发展较为平缓，缺少爆点
  评分：7.0分

五、总体评价
该IP整体质量中上，情感真挚细腻，人物关系刻画到位，但创新性和戏剧性相对不足。相比同类都市情感IP，竞争力中等偏上，适合作为中等成本的剧集项目。

总评分：7.6分

六、建议
该IP有一定改编潜力，但需要进行以下修改：
1. 增强情节的戏剧张力和冲突强度
2. 在人物设定上增加更多创新元素
3. 优化节奏，增加记忆点和爆点
4. 明确差异化定位，避免同质化
```

## 示例：评估玄幻小说IP

### 输入

```
请评估以下玄幻小说的IP价值：

小说名：《九天神皇》
作者：李四
类型：东方玄幻

核心设定：主角从废柴逆袭成为九天之上的神皇，历经重重磨难，最终登临巅峰

市场表现：
- 起点800万字作品
- 10万收藏
- 均订5万
- 评分7.8分

作者信息：
- 老牌作者，多部作品改编
- 之前改编作品表现中等
```

### 输出

```
【IP评估报告】

一、基本信息
- 故事梗概：废柴少年意外获得神秘传承，历经磨难，最终登临九天之上成为神皇
- 故事主题：逆袭、成长、力量与正义
- 人物关系：主角、师父、红颜知己、各类反派
- 市场表现：800万字，10万收藏，均订5万，评分7.8分
- 作者信息：老牌作者，多部作品有改编经历

二、市场潜力分析
- 受众适合度：精准贴合玄幻网文受众，男性为主
  评分：8.0分
- 讨论热度：经典逆袭题材，但观众审美疲劳，话题度有限
  评分：7.0分
- 稀缺性：废柴逆袭题材泛滥，稀缺性严重不足
  评分：6.5分
- 播放数据：网文基础扎实，但改编难度大，市场表现存疑
  评分：7.2分

三、创新属性分析
- 核心选点：废柴逆袭是最经典也最滥俗的设定
  评分：6.0分
- 故事概念：故事概念传统，缺乏独特记忆点
  评分：6.5分
- 故事设计：升级流结构完整，但套路化严重
  评分：6.8分

四、内容亮点分析
- 主题立意：励志成长主题正面但缺乏新意
  评分：7.0分
- 故事情境：世界观宏大但套路化，戏剧张力不足
  评分：6.8分
- 人物设定：主角人设脸谱化，配角功能性过强
  评分：6.5分
- 人物关系：人物关系模式化，缺乏深度
  评分：6.5分
- 情节桥段：情节发展可预期，缺少惊喜
  评分：6.5分

五、总体评价
该IP属于典型的玄幻升级流作品，网文基础尚可，但创新性严重不足，同质化问题突出。在当前影视市场环境下，改编为剧版的风险较大，建议谨慎考虑。如需开发，必须进行大幅度的创新改编。

总评分：6.8分

六、建议
1. 不建议直接按原著改编，必须进行创新性改编
2. 可以提取核心世界观，重新设计故事和人物
3. 建议增加差异化设定，避免同质化
4. 谨慎推进，建议先进行小成本的试水
```
