# 竖屏短剧评估专家 - 详细示例

## 示例：评估都市逆袭故事

### 输入

```
请评估以下故事改编为竖屏短剧的潜力：

故事梗概：
林浅是一名职场女性，在一家广告公司工作。她的上司王总是个势利眼，经常在公开场合羞辱她。一天，公司来了一个重要客户，王总想让林浅去陪客户喝酒，林浅拒绝后遭到更加严厉的羞辱。

林浅没有放弃，她努力工作，凭借自己的能力和创意，为公司赢得了一个重要项目。在项目发布会上，林浅作为项目负责人发言，王总也不得不认可她的能力。最终，林浅得到了公司的表彰和晋升。
```

### 输出

```
【竖屏短剧改编潜力评估报告】

【核心爽点】：评分：7.8
核心爽点：底层职场女性通过智慧和努力，打脸势利眼上司，最终逆袭成功的故事

分析与评估：该爽点具备一定的普世化程度，许多职场人都经历过类似的压抑和渴望被认可的情绪。但爽点呈现相对温和，缺少极致的戏剧冲突。

对标市场爆款：
- 优势：贴近现实，容易引起共鸣
- 不足：爽感不够强烈，缺少"打脸"的快感

优化建议：
1. 增加主角被羞辱的极致场景，为后续打脸蓄力
2. 让主角的反击更加迅速和直接，减少铺垫
3. 增加身份反转的震撼性（如：主角其实是隐藏大佬）

【故事类型】：评分：7.5
故事类型：都市职场逆袭+女性成长

分析与评估：使用的是经典逆袭模式，但创新性不足。套路相对传统，缺少独特记忆点。

对标市场爆款：
- 优势：类型明确，目标受众清晰
- 不足：缺少创新元素，同质化严重

优化建议：
1. 增加独特的设定元素（如：主角有特殊能力、特殊身份等）
2. 将经典套路极致化，做到同类题材中的极致
3. 融合热门元素（如：穿书、重生、系统等）

【人物设定】：评分：7.3
主要角色：
- 主角林浅：职场女性，坚韧独立
- 反派王总：势利眼上司，脸谱化反派

分析与评估：主角人设相对平淡，缺少极致的反差和标签化特点。反派过于脸谱化，缺少层次。

对标市场爆款：
- 优势：人物性格清晰
- 不足：缺少极致化设定，记忆点不强

优化建议：
1. 让主角的人设更加极致（如：超级社恐却必须社交、表面小白实则大佬等）
2. 增加主角的独特技能或特质
3. 让反派更加嚣张，增强打脸的爽感

【人物关系】：评分：7.2
主要人物关系：林浅↔王总（对立）、林浅↔同事（支持）

分析与评估：人物关系清晰但张力不足，缺少极致的对立和紧密的绑定。

对标市场爆款：
- 优势：关系结构简单清晰
- 不足：缺少戏剧张力和冲突强度

优化建议：
1. 增强主角与反派的直接对立和冲突
2. 增加人物关系的复杂度和层次
3. 让主角与反派的对抗更加直接和激烈

【情节桥段】：评分：7.0
情节桥段：主角被上司羞辱→主角努力工作→主角获得机会证明自己→主角成功打脸上司

分析与评估：情节发展相对平缓，缺少极致的情绪起伏和戏剧冲突。节奏不够紧凑，爽感释放不够强烈。

对标市场爆款：
- 优势：情节发展完整合理
- 不足：缺少爆点，悬念设置不足

优化建议：
1. 省略过程，直接切入矛盾核心
2. 每一集结尾设置强悬念钩子
3. 增加情节的戏剧性和视觉冲击力
4. 让打脸场面更加震撼和直接

【综合评估】：
该故事具备一定的改编潜力，整体质量中等偏上，但缺少爆款所需的极致性和戏剧性。在核心爽点、人物设定、情节桥段等方面都需要进行较大程度的优化，才能达到爆款竖屏短剧的标准。

核心风险点：同质化严重，缺少独特记忆点，爽感不足
核心机遇点：贴近现实，受众基础明确，易于产生共鸣

建议：谨慎推进，建议进行大幅度的优化后再考虑开发
```

## 示例：评估甜宠爱情故事

### 输入

```
请评估以下故事的改编潜力：

故事梗概：
富二代顾言隐瞒身份体验生活，成为一名外卖员。他遇到了被前男友羞辱的苏雅，帮助她解围。苏雅以为顾言只是个普通的外卖员，两人渐生情愫。

前男友带着新女友出现，再次羞辱苏雅和顾言。关键时刻，顾言的身份被揭露，他是顾氏集团的董事长。前男友和女友被赶出餐厅，顾言和苏雅终成眷属。
```

### 输出

```
【竖屏短剧改编潜力评估报告】

【核心爽点】：评分：8.2
核心爽点：隐藏大佬身份的总裁，在关键时刻揭露身份，狠狠打脸势利眼前男友

分析与评估：该爽点具备极强的普世化程度，"身份反转"是竖屏短剧最经典也最有效的爽点之一。身份反差带来的爽感强烈且直接。

对标市场爆款：
- 优势：经典爽点，受众基础广泛，爽感强烈
- 不足：套路较为常见，需要在细节上创新

优化建议：
1. 增强身份反差的极致性（如：让前男友的羞辱更加嚣张）
2. 让身份揭露的方式更加戏剧化、可视化
3. 增加铺垫阶段的甜度，强化观众期待

【故事类型】：评分：8.0
故事类型：甜宠+打脸+身份反差

分析与评估：融合了甜宠、打脸、身份反差三大热门元素，类型定位精准。虽然是经典模式，但执行得当可以成为爆款。

对标市场爆款：
- 优势：元素组合经典，目标受众明确
- 不足：需要避免同质化，做出差异化

优化建议：
1. 在甜宠桥段上做出新意，避免套路化
2. 强化身份反差的戏剧性
3. 增加独特的记忆点（如：特殊的相遇方式、独特的互动模式）

【人物设定】：评分：7.8
主要角色：
- 主角顾言：隐藏身份的富二代总裁，外表普通实则大佬
- 主角苏雅：被羞辱的普通女性，单纯善良
- 反派前男友：势利眼，脸谱化反派

分析与评估：主角的人设具备了反差感，但还可以更加极致。前男友的人设过于脸谱化，可以增加层次。

对标市场爆款：
- 优势：人物关系清晰，有反差感
- 不足：人设可以更加极致化、标签化

优化建议：
1. 让顾言的"隐藏身份"更加极致（如：超级抠门、超级节约）
2. 让苏雅有更鲜明的标签（如：吃货、社恐等）
3. 让前男友的嚣张更加极致，为打脸蓄力

【人物关系】：评分：7.5
主要人物关系：顾言↔苏雅（甜宠）、顾言↔前男友（对立）、苏雅↔前男友（前任关系）

分析与评估：人物关系清晰，有甜宠线和对立线，结构完整。但关系的极致性和戏剧性还可以增强。

对标市场爆款：
- 优势：关系结构完整，有多条线索
- 不足：极致性和戏剧张力还可以增强

优化建议：
1. 增强顾言和苏雅之间的甜度
2. 增强前男友与主角之间的对立强度
3. 可以增加一条副线（如：顾言家族的压力）

【情节桥段】：评分：7.8
情节桥段：苏雅被前男友羞辱→顾言帮助解围→两人渐生情愫→前男友再次出现→顾言身份揭露→打脸成功

分析与评估：情节结构完整，有起承转合。但情节的极致性和悬念设置还可以优化。

对标市场爆款：
- 优势：情节完整，节奏紧凑
- 不足：悬念设置不足，情绪极致性还可以增强

优化建议：
1. 每一集结尾设置强悬念钩子
2. 让身份揭露的方式更加戏剧化、可视化
3. 增加甜宠桥段的甜蜜度

【综合评估】：
该故事具备较强的改编潜力，类型定位精准，爽点明确。在人物设定、情节桥段等方面进行优化后，有成为爆款的潜力。整体质量中上，建议推进开发。

核心风险点：同质化风险，需要在细节上做出差异化
核心机遇点：经典爽点，受众基础广泛，易于成为爆款

建议：建议推进开发，同时进行优化
```
