# 故事大纲评估专家 - 详细示例

## 示例 1: 都市爱情大纲评估

### 输入

```
请评估以下故事大纲：

【大纲内容】
林浅和陆远在广告公司从竞争对手发展为恋人。剧本刻画了两人在工作中从针锋相对到相互欣赏的过程。前女友的出现制造了考验，但两人最终克服阻碍走到一起。
```

### 输出

```
【故事大纲评估报告】

一、大纲结构分析
- 结构完整性：完整，包含开端、发展、高潮、结局
- 情节清晰度：清晰，情节发展脉络明确
- 逻辑连贯性：逻辑连贯，因果关系合理

二、内容评估
- 故事创意：都市爱情题材，创意一般
- 人物设定：主角设定清晰，配角单薄
- 情节设计：情节发展合理，但缺少惊喜
- 情感线索：情感线清晰，但张力不足

三、改编潜力评估
- 影视改编价值：中等，适合网络剧或短片
- 商业化潜力：中等，受众定位准确但竞争激烈
- 开发建议：建议增加戏剧冲突和惊喜元素

【总体评分】：7.5/10

【改进建议】
1. 增加配角层次，丰富人物关系
2. 增强情节的戏剧性和冲突强度
3. 在情感线上增加更多波折
```

## 示例 2: 古装宫斗大纲评估

### 输入

```
请评估以下故事大纲：

【大纲内容】
苏婉从礼部尚书嫡女到皇后的历程。入宫后得宠，后被皇后陷害打入冷宫。在冷宫中学到宫廷斗争技巧，最终反击成为皇后。
```

### 输出

```
【故事大纲评估报告】

一、大纲结构分析
- 结构完整性：完整，遵循经典的宫斗剧结构
- 情节清晰度：清晰，情节转折点明确
- 逻辑连贯性：逻辑严密，前后呼应

二、内容评估
- 故事创意：古装宫斗，是成熟题材
- 人物设定：主角成长弧线完整
- 情节设计：情节设计精妙，张弛有度
- 情感线索：情感线复杂丰富

三、改编潜力评估
- 影视改编价值：高，适合电视剧开发
- 商业化潜力：高，有稳定受众基础
- 开发建议：建议保持现有质量，适当增加新意

【总体评分】：8.5/10

【改进建议】
1. 在经典框架下增加一些新意
2. 丰富配角人物，增加层次
3. 在细节上做精做细
```

## 示例 3: 职场励志大纲评估

### 输入

```
请评估以下故事大纲：

【大纲内容】
苏雅从普通白领到项目经理的励志故事。面对上司打压，苏雅没有放弃，在顾言的帮助下展现能力，最终获得晋升。
```

### 输出

```
【故事大纲评估报告】

一、大纲结构分析
- 结构完整性：完整，有起承转合
- 情节清晰度：清晰，情节发展合理
- 逻辑连贯性：逻辑连贯，因果关系清晰

二、内容评估
- 故事创意：职场励志，较为常见
- 人物设定：主角真实可信
- 情节设计：情节发展合理但缺少惊喜
- 情感线索：情感线较为简单

三、改编潜力评估
- 影视改编价值：中等，适合短视频或微电影
- 商业化潜力：中等，有现实意义但创新不足
- 开发建议：建议增加戏剧冲突和反转

【总体评分】：7.0/10

【改进建议】
1. 增加更多戏剧冲突
2. 设计更多反转和惊喜
3. 丰富人物关系
4. 在情节上增加爆点
```
