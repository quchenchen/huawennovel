# 文本截断工具 - 详细示例

## 示例 1: 在句号处截断

### 输入

```
文本="这是一个示例文本。这是第二句话。这是第三句话。", 最大长度=20
```

### 输出

```
原始长度：32
截断后长度：10
截断位置：在第一句句号处

截断后的文本：
这是一个示例文本。
```

## 示例 2: 在换行处截断

### 输入

```
文本="第一段内容
第二段内容
第三段内容", 最大长度=30
```

### 输出

```
原始长度：29
截断后长度：15
截断位置：在第一段换行处

截断后的文本：
第一段内容
```

## 示例 3: 在指定长度处截断

### 输入

```
文本="这是一个很长很长的句子里面没有任何标点符号所以只能在指定长度处进行截断", 最大长度=25
```

### 输出

```
原始长度：43
截断后长度：25
截断位置：在指定长度处截断

截断后的文本：
这是一个很长很长的句子里
```

## 示例 4: 带截断标记

### 输入

```
文本="这是一段很长的文本内容，需要被截断。这是第二段内容，也需要被截断。这是第三段内容，同样需要被截断。", 最大长度=50, 截断标记="..."
```

### 输出

```
原始长度：59
截断后长度：53
截断位置：在第二段句号处

截断后的文本：
这是一段很长的文本内容，需要被截断。这是第二段内容，也需要被截断。...
```

## 示例 5: 段落文本截断

### 输入

```
文本="故事讲述了一个年轻女孩从乡村来到城市打拼的故事。她经历了许多困难和挫折，但从未放弃自己的梦想。最终，她通过自己的努力和坚持，实现了人生的价值。这个故事告诉我们，只要坚持不懈，就一定能够成功。", 最大长度=100
```

### 输出

```
原始长度：125
截断后长度：100
截断位置：在第三句句号处

截断后的文本：
故事讲述了一个年轻女孩从乡村来到城市打拼的故事。她经历了许多困难和挫折，但从未放弃自己的梦想。最终，她通过自己的努力和坚持，实现了人生的价值。
```
