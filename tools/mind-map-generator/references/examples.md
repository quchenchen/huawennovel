# 思维导图生成专家 - 详细示例

## 说明

本技能调用 `generateTreeMind` 工具生成思维导图，返回包含图片 URL 和编辑链接的 JSON 响应。

## 示例 1: 故事五元素思维导图

### 输入

```
为以下故事生成五元素思维导图：

林浅和陆远在广告公司从竞争对手发展为恋人的故事。林浅是28岁的策划经理，独立自强。陆远是创意总监，才华横溢。两人在竞标中初识，在合作中相知，最终走到一起。
```

## 示例 2: 人物关系思维导图

### 输入

```
为以下故事生成人物关系思维导图：

苏婉入宫后与皇上、皇后、废太妃产生复杂关系。苏婉得宠，皇后嫉妒陷害。苏婉被打入冷宫，结识废太妃。太妃教导苏婉宫廷斗争技巧。最终苏婉成功反击。
```
